# AI 规则助手

这是一个面向万智牌规则问答与知识检索的纯前端对话页面（Tactical Judge Mode）。可以在浏览器中直接连接大语言模型 API 辅助判罚。

## 核心特性

- **极简式工作流**：支持左右侧边栏独立收缩与独立滚动，支持沉浸式的大段对话。
- **动态规则联动**：AI 给出带规则号的回复后，右侧会自动解析并呈现最新版的 Markdown 规则原文。
- **便携式配置**：集成了连接配置、系统提示词定制、消息记录检索（支持规则号及收藏过滤）。
- **单卡互动支持**：聊天中可使用 `[[卡牌名]]` 语法，或通过左侧强制配置自动在发送前抓取卡牌细节。

## 文件结构

- `index.html`：页面结构
- `styles.css`：全局视觉、响应式布局与折叠面板样式
- `app.js`：聊天控制、本地存储、渲染以及 API 调用逻辑

## 在线使用

**[点击这里体验](https://dingqiuwu.github.io/magic-comp-rules-zh-cn-agent/web-gemini-chat/)**

1. 在左侧面板填入对应的 API URL 与 Key (支持兼容 OpenAI 格式的接口)。
2. 选择合适的模型并调整提示词即可发送消息。
3. `API Key` 和聊天记录仅保存在您的浏览器 `localStorage` 中。

> **注意**：
> - 右侧的“规则引用原文”需请求此仓库内的 `.md` 规则文件，网页必须在 HTTP/HTTPS 环境下运行，不可直接双击打开本地 HTML。
> - 生产环境建议自行部署后端代理，以避免在客户端直接暴露密钥。
